Acte VI


Ce site web est une application mobile qui a été crée dans mon ancienne école Hetic, avec les langages HTML5, CSS et Javascript. Nous avons utilisés les logiciels Adobe pour la création de notre monde. Nous devions choisir un thème et crée un jeu dans lequel l 'utilisateur puisse naviguer et jouer. Nous avons choisi le thème des gilets jaunes car c est le quotidien. Le but du héros est de partir d 'un point de départ, récupérer des d 'autres gilets jaunes, tout en gagnant des points et arriver a l 'Arc de Triomphe. Attention , ils doit faire gare aux différents obstacles qui se met sur son chemin et éviter les policiers.'

  2. - Date: Decembre 2018
  3. - Compétences acquises : Javascript
